# Iris Flower Classification

CodSoft Data Science Internship - Task 3

## Objective
Develop a machine learning model to classify iris flowers into different species based on their sepal and petal measurements.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Dataset
https://raw.githubusercontent.com/uiuc-cse/data-fa14/gh-pages/data/iris.csv

## Model
K-Nearest Neighbors (KNN)

## Features Used
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Classes
- Setosa
- Versicolor
- Virginica
